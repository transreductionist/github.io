# github.io
Mind Prints

These packages need to be installed:
- Make sure that sphinx-rtd-theme is installed:  
  - pip install sphinx-rtd-theme
- Make sure the sphinxcontrib.gtagjs is installed:  
  - pip install sphinxcontrib.gtagjs

Change directory to:
- cd /Users/aaron/devwork/github.io

Run the command:
- sphinx-build --builder html -write-all --fresh-env mind_prints docs

After successful build open the index HTML file located at:
- /Users/aaron/devwork/github.io/docs/index.html

To visit site go to:
- fourfold-root.com

Use Google Analytics
